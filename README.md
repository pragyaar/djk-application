## The EXPO Lead Automation

Takes a CSV of EXPO leads and generates personalised follow-up email sequences — scored by engagement level and seniority, segmented by product interest, with a 14-day send calendar.

```bash
python3 expo_lead_automation.py
```

Outputs: `email_sequences.json`, `followup_calendar.csv`, `summary_report.txt`
